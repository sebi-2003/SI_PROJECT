This section presents a collection of projects focused on the study and application of system identification techniques, 
particularly Linear Regression and ARX (AutoRegressive with eXogenous inputs) models. 
Within these projects, various identification methods were explored and applied to provided datasets in order to analyze their behavior, performance, and limitations.

The work includes both theoretical understanding and practical implementation, highlighting the process of modeling dynamic systems,
comparing different approaches, and evaluating their effectiveness. 
Additionally, simplified versions of these methods were applied to small-scale systems, 
demonstrating the ability to translate theoretical concepts into practical engineering solutions.
